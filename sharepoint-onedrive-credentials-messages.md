# <a name="credential-messages-in-sharepoint-or-onedrive"></a>Ủy nhiệm thư trong SharePoint hoặc OneDrive

<p>Người dùng có thể được theo định kỳ được nhắc bởi văn phòng cho thông tin đăng nhập OneDrive hoặc SharePoint Online khi cố gắng để mở tài liệu.</p> <p>Sau đây là một số ví dụ về các ủy nhiệm thư:</p> <p>&bull;<span style="mso-tab-count: 1;"> &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; </span>Đăng nhập bằng tài khoản tổ chức của bạn</p> <p>&bull;<span style="mso-tab-count: 1;"> &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; </span>Thông tin đăng nhập được yêu cầu</p> <p>&bull;<span style="mso-tab-count: 1;"> &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; </span>Chúng tôi là không thể kết nối ngay bây giờ, hãy kiểm tra kết nối mạng của bạn.</p> <p>Nếu bạn đang gặp những triệu chứng này, hãy thử các bước trong <a href="https://support.microsoft.com/en-us/help/2913639/office-applications-periodically-prompt-for-credentials-to-sharepoint">bài viết này</a>.</p> <p>Ngoài ra, nếu bạn đang sử dụng đăng nhập đơn (SSO) xác thực, xin vui lòng tham khảo <a href="https://support.microsoft.com/en-us/help/4025962/cant-sign-in-after-update-to-office-2016-build-16-0-7967-on-windows-10">bài viết này</a> để thay thế.</p> <p>&nbsp;</p>
