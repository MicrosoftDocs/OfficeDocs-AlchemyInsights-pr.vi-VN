# <a name="migrate-files-from-dropbox"></a>Di chuyển tệp từ Dropbox

Để biết thêm thông tin, [hãy xem di chuyển Dropbox di Microsoft 365 với Trình quản lý Di chuyển](/sharepointmigration/mm-dropbox-overview).
