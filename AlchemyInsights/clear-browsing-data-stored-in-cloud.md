# <a name="clear-the-browsing-data-stored-in-the-cloud-if-youve-turned-on-sync-or-personalization"></a>Xóa dữ liệu duyệt web được lưu trữ trong đám mây (nếu bạn đã bật đồng bộ hoặc cá nhân hóa)

Để xem, quản lý hoặc xóa bỏ dữ liệu mà bạn đã lưu trong đám mây, hãy dùng bảng điều khiển quyền riêng tư. Dữ liệu bạn xóa bằng cách sử dụng bảng điều khiển sẽ không bị xóa khỏi thiết bị của bạn. Để biết chi tiết, hãy xem [xem dữ liệu của bạn trên bảng điều khiển quyền riêng tư](https://support.microsoft.com/windows/view-your-data-on-the-privacy-dashboard-03d3e27f-1981-5ff4-ba1c-d6b1031ae433).
