# <a name="public-folder-access-from-mobile-devices"></a>Truy cập thư mục công cộng từ thiết bị di động

Nếu bạn muốn có chức năng tương tự như thư mục công cộng và có nội dung có thể truy cập trên thiết bị di động, hãy tham khảo [Tìm hiểu về Microsoft 365 Groups để](https://support.office.com/article/learn-about-office-365-groups-b565caa1-5c40-40ef-9915-60fdb2d97fa2) có một lựa chọn tốt hơn.
