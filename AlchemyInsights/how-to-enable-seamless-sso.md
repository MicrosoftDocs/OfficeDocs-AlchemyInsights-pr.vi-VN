# <a name="how-to-enable-seamless-sso"></a>Làm thế nào để kích hoạt liền mạch SSO

Bật SSO liền mạch thông qua [AZURE AD kết nối](https://docs.microsoft.com/azure/active-directory/connect/active-directory-aadconnect).
  
Nếu bạn đang cài đặt mới Azure AD kết nối, chọn [đường dẫn cài đặt tuỳ chỉnh](https://docs.microsoft.com/azure/active-directory/connect/active-directory-aadconnect-get-started-custom). Tại trang **đăng nhập người dùng** , hãy chọn tùy chọn **bật đăng nhập đơn** .
  
Để xác minh rằng bạn đã kích hoạt liền mạch SSO đúng:
  
1. Đăng nhập vào [Trung tâm quản trị Azure Active Directory](https://aad.portal.azure.com) là một quản trị toàn cầu.

2. Chọn **Azure Active Directory** trong ngăn bên trái.

3. Xác minh rằng đăng nhập đơn liền mạch được **kích hoạt**.

Để tìm hiểu thêm, hãy xem [Azure Active Directory liền mạch đăng nhập đơn: bắt đầu nhanh](https://docs.microsoft.com/azure/active-directory/connect/active-directory-aadconnect-sso-quick-start).
