# <a name="access-services"></a>Dịch vụ truy cập

Tính năng truy cập Dịch vụ SharePoint trực tuyến đang được nghỉ hưu. Mới truy cập Dựa trên ứng dụng web và cơ sở dữ liệu web không còn có thể được tạo ra, và các ứng dụng web hiện tại và cơ sở dữ liệu web sẽ được tắt bởi Tháng tư, 2018. Truy cập cơ sở dữ liệu máy tính để bàn (. accdb) không bị ảnh hưởng bởi thay đổi này. Chúng tôi khuyên Microsoft PowerApps là một thay thế cho truy cập ứng dụng web và cơ sở dữ liệu web. 
  
[Tìm hiểu về Microsoft PowerApps](https://powerapps.microsoft.com/)
