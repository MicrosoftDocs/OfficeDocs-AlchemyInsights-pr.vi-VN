# <a name="webpage-html-web-resources-limitations"></a>Giới hạn tài nguyên web của trang web (HTML)

* Tài nguyên web HTML không thể chứa bất kỳ mã nào phải được thực hiện trên máy chủ. ASP.NET trang không thể tải lên dưới dạng tài nguyên web HTML.

* Tài nguyên web HTML chỉ có thể chấp nhận một số giới hạn các tham số chuỗi truy vấn. Thông tin thêm: [chuyển tham số cho tài nguyên web HTML](https://docs.microsoft.com/dynamics365/customer-engagement/developer/webpage-html-web-resources#BKMK_PassingParametersToWebResources)
