# <a name="webpage-html-web-resources-limitations"></a>Các giới hạn về tài nguyên web (HTML) trang web

* Tài nguyên web HTML không thể chứa bất kỳ mã nào phải được thực thi trên máy chủ. Không thể tải lên trang ASP.NET dưới dạng tài nguyên web HTML.

* Tài nguyên web HTML chỉ có thể chấp nhận số lượng tham số chuỗi truy vấn hạn chế. Thông tin thêm: [gởi tham số vào tài nguyên web HTML](https://docs.microsoft.com/dynamics365/customer-engagement/developer/webpage-html-web-resources#BKMK_PassingParametersToWebResources)
