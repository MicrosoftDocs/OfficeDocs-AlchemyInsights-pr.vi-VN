# <a name="create-an-allow-list-of-endpoints-for-microsoft-edge"></a>Tạo danh sách cho phép các điểm cuối cho Microsoft Edge

Để có một mảng hoàn chỉnh các tính năng của Microsoft Edge theo ý bạn, bạn phải kết nối với Internet. Ngoài ra, để cho phép trình duyệt liên lạc qua tường lửa và các cơ chế bảo mật khác trên Internet, bạn phải thêm bộ URL tên miền vào [danh sách điểm cuối của](https://go.microsoft.com/fwlink/?linkid=2135054)nó.
