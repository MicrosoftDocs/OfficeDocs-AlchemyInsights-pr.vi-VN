# <a name="hub-site-navigation"></a>Trung tâm trang web chuyển hướng

Ở trên cùng của một trang web trung tâm và bất kỳ trang web liên kết trên trang web chuyển hướng, là thanh điều hướng trang web của Trung tâm. Nếu bạn là chủ sở hữu trang web trung tâm, bạn có thể tùy chỉnh [thanh điều hướng](https://support.office.com/en-us/article/customize-the-navigation-on-your-sharepoint-site-3cd61ae7-a9ed-4e1e-bf6d-4655f0bf25ca#hubnav). 

**Lưu ý** Đó khuyến cáo để chọn một trang web thông tin liên lạc, hoặc một nhóm trang web bằng cách sử dụng các mẫu hiện đại như Trung tâm trang web của bạn. Nếu bạn sử dụng một trang web cổ điển đội ngũ, định vị trung tâm sẽ chỉ xuất hiện trên các trang web hiện đại và thiết lập trang web trung tâm sẽ chỉ xuất hiện trên trang hiện đại.
