# <a name="sharepoint-hub-site-navigation"></a>SharePoint Trung tâm điều hướng trang web

Ở đầu trang Trung tâm và bất kỳ trang web nào được liên kết, phía trên điều hướng trang web, là thanh điều hướng trang web trung tâm. Nếu bạn là chủ sở hữu trang web trung tâm, bạn có thể tùy chỉnh [thanh điều hướng](https://support.office.com/article/customize-the-navigation-on-your-sharepoint-site-3cd61ae7-a9ed-4e1e-bf6d-4655f0bf25ca#hubnav). 

**Lưu ý:** Chúng tôi khuyên bạn nên chọn một trang web liên lạc, hoặc một trang web nhóm bằng cách sử dụng mẫu hiện đại làm trang web trung tâm của bạn. Nếu bạn sử dụng trang web nhóm cổ điển, điều hướng trung tâm sẽ chỉ xuất hiện trên các trang hiện đại và cài đặt trang web trung tâm sẽ chỉ xuất hiện trên trang hiện đại.
