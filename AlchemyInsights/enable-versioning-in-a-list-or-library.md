# <a name="enable-versioning-for-a-list-or-library"></a>Bật Phiên bản cho danh sách hoặc thư viện

1. Đi tới danh sách hoặc thư viện.
    
2. Bấm **cài đặt**, và sau đó bấm cài đặt **Thư viện** hoặc **danh sách cài đặt**.
    
3. Nhấp vào **cài đặt versioning**.
    
4. Chọn cài đặt bạn muốn, sau đó bấm **OK**.
