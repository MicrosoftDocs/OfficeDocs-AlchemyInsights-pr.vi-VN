# <a name="enable-versioning-for-a-list-or-library"></a>Bật lập phiên bản cho một danh sách hoặc thư viện

1. Đi đến danh sách hoặc thư viện.
    
2. Bấm **thiết đặt**, sau đó bấm Thiết đặt **Thư viện** hoặc **thiết đặt danh sách**.
    
3. Bấm vào **thiết đặt lập**Phiên bản.
    
4. Chọn thiết đặt bạn muốn, rồi bấm **OK**.
