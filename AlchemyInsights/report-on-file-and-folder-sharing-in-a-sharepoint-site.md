# <a name="report-on-file-and-folder-sharing-in-a-sharepoint-site"></a>Báo cáo chia sẻ tệp và cặp trong một trang web SharePoint

Bạn có thể tạo tệp CSV của mọi tệp, người dùng, quyền và liên kết duy nhất trên trang web SharePoint hoặc thư viện OneDrive đã cho. Điều này có thể giúp bạn hiểu cách chia sẻ đang được sử dụng và liệu bất kỳ tệp hoặc thư mục nào đang được chia sẻ với khách. Để biết thêm thông tin, hãy xem [báo cáo chia sẻ tệp và cặp trong một trang web SharePoint](https://docs.microsoft.com/sharepoint/sharing-reports).
