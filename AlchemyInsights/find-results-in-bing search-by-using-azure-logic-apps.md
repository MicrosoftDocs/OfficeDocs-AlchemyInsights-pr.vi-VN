# <a name="find-results-in-bing-search-by-using-azure-logic-apps"></a>Tìm kiếm kết quả trong Bing Search bằng cách dùng ứng dụng Azure logic

Bằng cách sử dụng tìm kiếm Bing từ bên trong một ứng dụng logic, bạn có thể tự động xử lý kết quả tìm kiếm và thực hiện những kết quả này sẵn dùng cho những hành động khác. Ví dụ, bạn có thể tìm thấy các mục tin tức và sau đó có Twitter đăng chúng như tweets trong nguồn cấp dữ liệu của bạn.

Để tìm hiểu thêm, hãy xem [Tìm kết quả trong tìm kiếm Bing bằng cách dùng ứng dụng Azure logic](https://go.microsoft.com/fwlink/?linkid=2151928).
