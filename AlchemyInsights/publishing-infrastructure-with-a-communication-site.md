# <a name="publishing-infrastructure-with-a-communication-site"></a>Cơ sở hạ tầng xuất bản với một trang web thông tin liên lạc

Cho phép các tính năng SharePoint Server cơ sở hạ tầng xuất bản là **không được hỗ trợ** với các trang web thông tin liên lạc. Vui lòng cung cấp thông tin phản hồi của bạn tại [UserVoice](https://go.microsoft.com/fwlink/?linkid=2047322&amp;clcid=0x409). 
  
Hãy xem [kích hoạt tính năng xuất bản](https://support.office.com/en-us/article/Enable-publishing-features-479677A6-8B33-4AC7-907D-071C1C7E4518) cho biết thêm thông tin.
