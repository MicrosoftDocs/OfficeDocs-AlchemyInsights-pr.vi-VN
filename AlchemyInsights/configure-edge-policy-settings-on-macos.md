# <a name="use-jamf-to-configure-microsoft-edge-policy-settings-on-macos"></a>Dùng Jamf để cấu hình các thiết đặt chính sách của Microsoft Edge trên macOS

Để đặt cấu hình cho Microsoft Edge (ổn định kênh 81 trở lên) trên macOS, hãy dùng tệp kê khai Microsoft Edge Policy trên Jamf Pro 10,19.

Để biết thêm thông tin, hãy xem đặt [cấu hình thiết đặt chính sách của Microsoft Edge trên macOS với Jamf](https://go.microsoft.com/fwlink/?linkid=2134761).
