# <a name="create-a-site"></a>Tạo một trang web

Bạn có thể tạo ra các loại khác nhau của các trang web trong SharePoint Online. Để tìm hiểu làm thế nào để làm điều này trong Trung tâm quản trị SharePoint mới, hãy xem [tạo một trang web](https://go.microsoft.com/fwlink/?linkid=866295). Để xem thông tin về cho phép người dùng tạo ra các trang web riêng của họ, hãy xem [quản lý tạo ra trang web trong SharePoint Online](https://go.microsoft.com/fwlink/?linkid=866296).
 
Để xem thông tin về các trang web mới, hãy xem [nhóm kết nối nhóm các trang web](https://go.microsoft.com/fwlink/?linkid=866292) hoặc [các trang web thông tin liên lạc](https://go.microsoft.com/fwlink/?linkid=866294).
