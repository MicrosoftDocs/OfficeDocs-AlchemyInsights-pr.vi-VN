# <a name="copy-and-paste-in-outlookcom"></a>Sao chép và dán trong Outlook.com

Không có sao chép hoặc dán các nút trong Outlook.com, nhưng bạn có thể sử dụng phím tắt:

**Trên một máy tính:** CTRL + C để sao chép, CTRL + X để cắt và CTRL + V để dán.

**Trên máy Mac:** CMD + C để sao chép, CMD + X để cắt và CMD + V để dán.
