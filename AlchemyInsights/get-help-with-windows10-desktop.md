# <a name="get-help-with-windows-10-desktop"></a>Nhận trợ giúp về máy tính chạy Windows 10

- Để tạo lối tắt màn hình nền, hãy xem [các ứng dụng và thư mục ghim vào màn hình nền hoặc thanh tác vụ](https://support.microsoft.com/windows/pin-apps-and-folders-to-the-desktop-or-taskbar-f3c749fb-e298-4cf1-adda-7fd635df6bb0).

- Để thực hiện các biểu tượng hoặc văn bản lớn hơn trong Windows 10, hãy xem mục [thay đổi kích cỡ của các biểu tượng trên màn hình nền của bạn](https://support.microsoft.com/windows/change-the-size-of-your-desktop-icons-85a9d341-2a4f-3d96-c796-ae116a187211).

- Để hiện các biểu tượng trên màn hình nền trong Windows 10, hãy xem [Hiển thị các biểu tượng trên máy tính trong Windows 10](https://support.microsoft.com/windows/show-desktop-icons-in-windows-10-c13270f0-3812-c71d-f27e-29aa32588b20).
