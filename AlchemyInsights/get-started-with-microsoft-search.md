# <a name="get-started-with-microsoft-search"></a>Bắt đầu với Microsoft Search

Sử dụng Microsoft Search để tìm mọi người, tệp, biểu đồ tổ chức, site, câu trả lời cho câu hỏi thường gặp và nội dung khác có thể giúp bạn hoàn thành các tác vụ trong ứng dụng mà bạn đang sử dụng.

Để biết thêm thông tin, hãy xem [tổng quan về Microsoft Search](https://docs.microsoft.com/microsoftsearch/overview-microsoft-search).
