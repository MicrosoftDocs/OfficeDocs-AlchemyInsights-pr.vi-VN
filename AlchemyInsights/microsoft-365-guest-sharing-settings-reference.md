# <a name="microsoft-365-guest-sharing-settings-reference"></a>Microsoft 365 khách chia sẻ cài đặt tham khảo

Bài viết sau đây cung cấp tham chiếu cho các cài đặt khác nhau có thể ảnh hưởng đến việc chia sẻ với những người bên ngoài tổ chức của bạn cho các khối lượng công việc Microsoft 365: teams, nhóm Microsoft 365, SharePoint và OneDrive. Các thiết đặt này nằm trong Trung tâm quản trị Azure Active Directory, Microsoft 365, nhóm và SharePoint.

- [Microsoft 365 khách chia sẻ cài đặt tham khảo](https://docs.microsoft.com/microsoft-365/solutions/microsoft-365-guest-settings?view=o365-worldwide)
