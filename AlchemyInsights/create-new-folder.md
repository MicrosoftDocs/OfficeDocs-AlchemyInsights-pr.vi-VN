# <a name="create-a-new-folder-in-outlookcom"></a>Tạo thư mục mới trong Outlook.com

1. Bên dưới **thư mục** trong ngăn dẫn hướng, hãy chọn **thư mục mới**.

2. Nhập tên cho thư mục, rồi nhấn Enter.

Để tìm hiểu thêm, hãy xem [tạo thư mục trong Outlook.com](https://support.office.com/article/5fa8de74-3562-4729-ac1d-5599f470b25a?wt.mc_id=Office_Outlook_com_Alchemy).
