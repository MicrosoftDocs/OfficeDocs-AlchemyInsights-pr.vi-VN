# <a name="create-a-new-folder"></a>Tạo một thư mục mới

1. Bên dưới **thư mục** trong ngăn dẫn hướng, chọn **thư mục mới**.

2. Gõ tên cho thư mục và nhấn Enter.

Để tìm hiểu thêm, hãy xem [tạo một thư mục trong Outlook.com](https://support.office.com/article/5fa8de74-3562-4729-ac1d-5599f470b25a).
