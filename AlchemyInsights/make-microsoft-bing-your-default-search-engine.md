# <a name="make-microsoft-bing-your-default-search-engine"></a>Đặt Microsoft Bing làm công cụ tìm kiếm mặc định của bạn

Để làm cho Bing công cụ tìm kiếm mặc định của bạn cho Microsoft Edge, Google Chrome hoặc Internet Explorer, hãy làm theo các hướng dẫn trong [thực hiện Bing công cụ tìm kiếm mặc định](https://go.microsoft.com/fwlink/?linkid=2148834).
