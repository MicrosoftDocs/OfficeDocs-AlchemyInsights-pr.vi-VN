# <a name="microsoft-stream-mobile-apps"></a>Ứng dụng Microsoft Stream dành cho thiết bị di động

Microsoft Stream sẵn dùng thông qua các ứng dụng iOS và Android dành cho thiết bị di động trên điện thoại và máy tính bảng để cho phép bạn xem video khi đang di chuyển.

- [Ứng dụng Microsoft Stream dành cho thiết bị di động](https://docs.microsoft.com/stream/mobile-apps-overview)
- [Tải ứng dụng dành cho thiết bị di động Microsoft Stream](https://docs.microsoft.com/stream/mobile-get-apps)
