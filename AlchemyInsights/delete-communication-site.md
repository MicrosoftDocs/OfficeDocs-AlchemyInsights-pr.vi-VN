# <a name="delete-a-communication-site"></a>Xóa một trang web thông tin liên lạc

1. Đi đến trang web.
    
2. Nhấp vào biểu tượng cài đặt ở phía trên bên phải.
    
3. Nhấp vào **thiết lập trang web**. Theo **Thông tin trang web**, nhấp vào **xóa trang web này**, và sau đó nhấp vào **xóa**.
