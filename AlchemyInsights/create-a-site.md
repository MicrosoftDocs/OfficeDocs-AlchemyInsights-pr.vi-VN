# <a name="create-a-site"></a>Tạo một trang web

- Truy cập [trang web trong Trung tâm quản trị Microsoft 365](https://portal.office.com/adminportal/home#/SitesList), và sau đó chọn **Thêm một trang web**. 
    
- Trong Trung tâm quản trị SharePoint mới, trên trang web, bấm **tạo**. 
    
**Lưu ý:** Bạn không thể tạo một trang web mới có cùng một URL với một trang web hiện có. 
  
Để biết thêm thông tin về cách tạo trang web, xem [tạo trang web](https://go.microsoft.com/fwlink/?linkid=866295).
  
Để biết thông tin về việc cho phép người dùng tạo trang web riêng của họ, xem [quản lý tạo trang web trong SharePoint Online](https://go.microsoft.com/fwlink/?linkid=866296).
