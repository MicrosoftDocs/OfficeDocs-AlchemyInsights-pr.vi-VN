# <a name="delete-a-classic-sharepoint-team-site-or-subsite"></a>Xóa một cổ điển SharePoint team trang web hay các subsite

1. Đi đến trang web.
    
2. Nhấp vào biểu tượng **cài đặt** ở phía trên bên phải. 
    
3. Nhấp vào **thiết lập trang web**. Theo **Trang web hoạt động**, nhấp vào **xóa trang web này**, và sau đó nhấp vào **xóa**.
