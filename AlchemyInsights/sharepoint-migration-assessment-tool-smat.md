# <a name="sharepoint-migration-assessment-tool-smat"></a>Công cụ đánh giá di chuyển SharePoint (SMAT)

Công cụ đánh giá di chuyển SharePoint (SMAT) là một dòng lệnh thực thi đơn giản mà sẽ quét nội dung của trang trại SharePoint của bạn để giúp xác định tác động của việc di chuyển máy chủ của bạn sang SharePoint Online với Microsoft 365.

[Tổng quan về công cụ đánh giá di chuyển SharePoint](https://docs.microsoft.com/sharepointmigration/overview-of-the-sharepoint-migration-assessment-tool)
