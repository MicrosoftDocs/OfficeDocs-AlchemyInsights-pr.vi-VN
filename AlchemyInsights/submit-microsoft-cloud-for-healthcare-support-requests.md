# <a name="submit-microsoft-cloud-for-healthcare-support-requests"></a>Gửi yêu cầu hỗ trợ của Microsoft Cloud for Healthcare

Microsoft Cloud for Healthcare khách hàng vui lòng gửi vé thông qua [Trung tâm quản trị Power Platform mới.](https://admin.powerplatform.microsoft.com/support?newTicket&product=Flow)

Cổng thông tin mới cung cấp đường dẫn trực tiếp hơn để hỗ trợ hỗ trợ được tối ưu hóa cho Microsoft Cloud for Healthcare.
