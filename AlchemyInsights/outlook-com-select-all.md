# <a name="how-to-select-all-in-outlookcom"></a>Cách "chọn tất cả" trong Outlook.com

Bạn chỉ có thể chọn tất cả nếu tất cả các thư đều hiển thị. Điều này là ngăn chặn các hành động tình cờ trên các thư mà bạn không thể nhìn thấy.

**Ctrl + bấm vào** chọn nhiều thư riêng lẻ.

**Shift + bấm vào** chọn tất cả thư giữa thư đầu tiên và cuối cùng mà bạn chọn.
