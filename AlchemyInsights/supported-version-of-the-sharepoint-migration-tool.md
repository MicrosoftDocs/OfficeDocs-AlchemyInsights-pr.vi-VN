# <a name="supported-version-of-the-sharepoint-migration-tool"></a>Hỗ trợ phiên bản công cụ di chuyển SharePoint



Hiện công cụ di chuyển SharePoint chỉ hỗ trợ di cư từ SharePoint 2013. Phiên bản 2010 và 2016 không được hỗ trợ tại thời gian này.
  
Để biết thêm thông tin, hãy xem [giới thiệu công cụ di chuyển SharePoint](https://go.microsoft.com/fwlink/?linkid=2044765&amp;clcid=0x409).
