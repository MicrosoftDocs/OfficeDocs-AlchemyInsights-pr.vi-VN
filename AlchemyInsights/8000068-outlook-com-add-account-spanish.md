# <a name="agregar-sus-otras-cuentas-de-correo-electrnico"></a>Agregar sus otras cuentas de Fernando electrónico

Cuando agregue otras cuentas de Fernando electrónico Outlook.com, puede usar Outlook.com para enviar y leer mensajes de Fernando electrónico de estas cuentas sin salir de Outlook.com. Cada cuenta que se agrega một Outlook.com se denomina una cuenta conectada. También puede [crear un bí danh de Fernando electrónico](https://support.office.com/es-es/article/agregar-o-quitar-un-alias-de-correo-electrónico-en-outlook-com-459b1989-356d-40fa-a689-8f285b13f1f2) que utiliza la misma Bandeja de entrada, enviar, elementos y lista de contactos como su cuenta de Outlook.com existente.
