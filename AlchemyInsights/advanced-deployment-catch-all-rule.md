# <a name="deploy-microsoft-365-apps"></a>Triển khai ứng dụng Microsoft 365

Để triển khai Microsoft 365 ứng dụng cho nhiều người dùng cuối qua công cụ triển khai Office, Microsoft Endpoint Configuration Manager, như một phần của ảnh đĩa hệ điều hành hoặc trong tình huống máy tính được chia sẻ bằng cách sử dụng Windows Server với dịch vụ máy tính để bàn từ xa (RDS) hoặc dịch vụ đầu cuối, xem xét sử dụng hướng dẫn từng bước trong **Microsoft 365 ứng**
  
Để bắt đầu, hãy đảm bảo rằng bạn đã đăng nhập với tư cách quản trị viên hoặc quản trị viên toàn cầu, sau đó bấm vào đây: [https://aka.ms/proplusguidance](https://aka.ms/proplusguidance) .
