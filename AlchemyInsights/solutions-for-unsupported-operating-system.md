# <a name="solutions-for-unsupported-operating-system"></a>Giải pháp cho "Không được hỗ trợ hệ điều hành"



Khi bạn cố gắng để cài đặt Office 365, bạn có thể thấy một thông báo lỗi **hệ điều hành không được hỗ trợ** nếu thiết bị của bạn không đáp ứng các [yêu cầu hệ thống](https://products.office.com/office-system-requirements).
  
Office 365 yêu cầu:
  
- PC: Windows 10, Windows 8.1, Windows 8, Windows 7 Service Pack 1, Windows Server 2016, Windows Server 2012 R2, Windows Server 2012, hoặc Windows Server 2008 R2
    
    HOẶC
    
- Mac OS X 10.10
    
Nếu nhận được thông báo này trong khi đang cố gắng để cài đặt Office trên điện thoại di động hoặc Chromebook, hãy xem [làm thế nào để cài đặt và chạy Microsoft Office trên Chromebook](https://support.office.com/article/32f14a23-2c1a-4579-b973-d4b1d78561ad?wt.mc_id=Alchemy_ClientDIA).
