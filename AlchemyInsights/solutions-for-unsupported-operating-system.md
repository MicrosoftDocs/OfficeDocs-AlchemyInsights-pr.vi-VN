PC: Windows 10, Windows 8.1, Windows 8, Windows 7 Service Pack 1, Windows Server 2016, Windows Server 2012 R2, Windows Server 2012, hoặc Windows Server 2008 R2
  
HOẶC
  
Mac OS X 10.10
