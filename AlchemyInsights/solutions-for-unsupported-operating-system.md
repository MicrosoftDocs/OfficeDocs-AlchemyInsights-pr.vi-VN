# <a name="solutions-for-unsupported-operating-system"></a>Giải pháp cho "hệ điều hành không hỗ trợ"

Khi bạn cố gắng cài đặt Microsoft 365, bạn có thể thấy một thông báo lỗi **hệ điều hành không được hỗ trợ** nếu thiết bị của bạn không đáp [ứng các yêu cầu hệ thống](https://products.office.com/office-system-requirements).
  
Microsoft 365 yêu cầu:
  
- PC: Windows 10, Windows 8,1, Windows 8, Windows 7 gói dịch vụ 1, Windows Server 2016, Windows Server 2012 R2, Windows Server 2012 hoặc Windows Server 2008 R2

    Hoặc

- Mac OS X 10,10

Nếu nhận được thông báo này trong khi cố cài đặt Office trên thiết bị di động hoặc Chromebook, xem [cách cài đặt và chạy Microsoft Office trên Chromebook](https://support.office.com/article/32f14a23-2c1a-4579-b973-d4b1d78561ad?wt.mc_id=Alchemy_ClientDIA).
