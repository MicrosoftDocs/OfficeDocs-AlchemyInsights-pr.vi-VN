# <a name="delete-a-sharepoint-site"></a>Xóa một trang web SharePoint

1. Trong Trung tâm quản trị mới SharePoint, trên trang web, nhấp vào hộp kiểm bên cạnh các trang web mà bạn muốn xóa.
    
2. Trên thanh công cụ ribbon, nhấp vào **xóa**.
