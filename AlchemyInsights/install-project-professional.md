# <a name="install-project-professional"></a>Cài đặt Project Professional

Xem mục [cài đặt dự án](https://support.office.com/article/install-project-7059249b-d9fe-4d61-ab96-5c5bf435f281) để tìm hiểu cách cài đặt phiên bản gói đăng ký của Project Online và phiên bản không phải đăng ký của Project 2019, 2016 hoặc 2013. 

Đối với Project 2010 hoặc 2007, hãy làm theo các bước trong [office 2010](https://support.office.com/article/install-office-2010-1b8f3c9b-bdd2-4a4f-8c88-aa756546529d)   hoặc [Office 2007](https://support.office.com/article/install-office-2007-88a8e329-3335-4f82-abb2-ecea3e319657). 

Dự án được bán với tư cách là một ứng dụng độc lập, vì vậy nếu bạn chưa có, [Hãy mua hoặc thử dự án](https://products.office.com/project).
