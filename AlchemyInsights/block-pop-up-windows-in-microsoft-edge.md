# <a name="block-pop-up-windows-in-microsoft-edge"></a>Chặn cửa sổ bật lên trong Microsoft Edge

1. Đi tới **... (Thiết đặt và nhiều hơn nữa)**  >  **Thiết đặt**  >  **Quyền** đối với site.
2. Chọn **cửa sổ bật lên và chuyển hướng**.
3. Bật **chặn**.
