# <a name="block-file-download-with-azure-ad-conditional-access"></a>Chặn tệp tải xuống với truy cập có điều kiện Azure AD

Chính sách phiên bảo mật ứng dụng đám mây cho phép bạn hạn chế một phiên dựa trên trạng thái thiết bị. Để thực hiện kiểm soát phiên sử dụng thiết bị của mình làm điều kiện, tạo cả chính sách truy cập có điều kiện và chính sách phiên.

- [Chặn tệp tải xuống với truy cập có điều kiện Azure AD](https://docs.microsoft.com/cloud-app-security/use-case-proxy-block-session-aad#create-a-block-download-policy-for-unmanaged-devices)
