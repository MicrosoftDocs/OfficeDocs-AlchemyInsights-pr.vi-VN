# <a name="configure-multi-factor-authentication"></a>Cấu hình xác thực đa yếu tố

Để đặt cấu hình MFA, hãy xem [thiết lập xác thực đa yếu tố](https://docs.microsoft.com/microsoft-365/admin/security-and-compliance/set-up-multi-factor-authentication).

Lưu ý rằng bạn cần phải là người quản trị toàn cầu để bật và cấu hình MFA.
