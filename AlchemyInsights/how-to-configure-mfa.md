# <a name="configure-multi-factor-authentication"></a>Đặt cấu hình xác thực đa yếu tố

Để đặt cấu hình MFA, [hãy xem mục Thiết lập xác thực đa yếu](/microsoft-365/admin/security-and-compliance/set-up-multi-factor-authentication) tố hoặc sử dụng hướng dẫn Trung tâm quản trị Microsoft 365 thiết [lập.](https://admin.microsoft.com/AdminPortal/Home?ref=/modernonboarding/mfasetupguide:)

**Lưu ý**: Bạn phải là người quản trị toàn cầu thì mới có thể bật và đặt cấu hình MFA.
