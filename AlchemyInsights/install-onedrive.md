# <a name="install-onedrive"></a>Cài đặt OneDrive

Khách hàng đồng bộ OneDrive là cài đặt sẵn với Windows 10. Để cài đặt OneDrive cho Windows 7 hoặc Mac, hoặc để có được các ứng dụng di động cho điện thoại hoặc máy tính bảng của bạn, đi đến [http://onedrive.com/download](http://onedrive.com/download).
