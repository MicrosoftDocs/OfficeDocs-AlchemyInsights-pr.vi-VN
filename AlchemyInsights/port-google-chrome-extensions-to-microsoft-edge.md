# <a name="port-google-chrome-extensions-to-microsoft-edge-chromium"></a>Các phần mở rộng của cổng Google Chrome vào Microsoft Edge (Crom)

Thật dễ dàng để [mở rộng cổng Google Chrome sang Microsoft Edge (Crom)](https://docs.microsoft.com/microsoft-edge/extensions-chromium/developer-guide/port-chrome-extension). Trong hầu hết các trường hợp, chỉ cần những thay đổi tối thiểu để chạy các phần mở rộng này trên Microsoft Edge.

Các API mở rộng và các phím Hiển thị được hỗ trợ bởi Google Chrome là mã tương thích với Microsoft Edge. Tuy nhiên, Microsoft Edge không hỗ trợ phần mở rộng API Chrome. GCM, Chrome. Identity. getAccounts, Chrome. Identity. getAuthToken và Chrome. instanceID.
