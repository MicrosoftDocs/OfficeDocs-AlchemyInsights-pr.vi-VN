# <a name="edit-a-read-only-file"></a>Sửa tệp chỉ đọc

Trong thư viện tài liệu, khi bạn bấm vào tên tài liệu, chương trình Office có thể đột ngột mở nó dưới dạng tệp chỉ đọc. Để mở ứng dụng này để chỉnh sửa, hãy bấm **mở** (ba chấm), rồi bấm **sửa.**
