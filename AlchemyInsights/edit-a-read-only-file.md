# <a name="edit-a-read-only-file"></a>Chỉnh sửa tệp chỉ đọc

Trong thư viện tài liệu, khi bạn bấm vào tên của tài liệu, chương trình Office bất ngờ có thể mở nó dưới dạng tệp chỉ-đọc. Để mở nó để chỉnh sửa, bấm **mở** (ba dấu chấm), và sau đó nhấp vào **chỉnh sửa.**
