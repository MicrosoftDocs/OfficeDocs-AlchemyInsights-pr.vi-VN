# <a name="edit-a-read-only-file"></a>Chỉnh sửa tệp chỉ-đọc

Trong thư viện tài liệu, khi bạn bấm vào tên của tài liệu, chương trình Office có thể bất ngờ mở nó như là một tệp chỉ-đọc. Để mở nó để chỉnh sửa, nhấp vào **mở** (ba dấu chấm), và sau đó nhấp vào **chỉnh sửa**
