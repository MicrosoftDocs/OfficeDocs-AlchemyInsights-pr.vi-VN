# <a name="recover-missing-or-deleted-site-collections"></a>Khôi phục bộ sưu tập trang web bị thiếu hoặc bị xóa

Làm thế nào để khôi phục bộ sưu tập trang web bị thiếu hoặc bị xóa.

[Khôi phục các trang web đã xóa](https://docs.microsoft.com/sharepoint/restore-deleted-site-collection)

[Khôi phục bằng mô-đun SPO PowerShell](https://support.office.com/article/Introduction-to-the-SharePoint-Online-Management-Shell-C16941C3-19B4-4710-8056-34C034493429)

Nếu bạn không thể tìm thấy trang web bạn có thể muốn [Tìm kiếm Nhật ký kiểm tra trong &amp; Office 365 Trung tâm bảo mật tuân thủ](https://docs.microsoft.com/office365/securitycompliance/search-the-audit-log-in-security-and-compliance).

**Lưu ý**: "hiện đại" nhóm site không hiển thị trong giao diện người dùng quản trị SharePoint **cổ điển** . Bạn có thể truy cập vào danh sách các trang web "hiện đại" từ [Trung tâm quản trị SharePoint mới](https://docs.microsoft.com/sharepoint/get-started-new-admin-center).
