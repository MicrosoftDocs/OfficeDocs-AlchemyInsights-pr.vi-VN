# <a name="recover-missing-or-deleted-site-collections"></a>Khôi phục thiếu hoặc xóa bộ sưu tập trang web

Làm thế nào để phục hồi bị mất hoặc bị xóa bộ sưu tập trang web.

[Khôi phục xóa các trang web](https://docs.microsoft.com/sharepoint/restore-deleted-site-collection)

[Khôi phục bằng cách sử dụng mô-đun SPO PowerShell](https://support.office.com/article/Introduction-to-the-SharePoint-Online-Management-Shell-C16941C3-19B4-4710-8056-34C034493429)

Nếu bạn không thể tìm thấy các trang web bạn có thể [Tìm kiếm Nhật ký kiểm tra trong Office 365 Security &amp; tuân thủ trung tâm](https://docs.microsoft.com/office365/securitycompliance/search-the-audit-log-in-security-and-compliance).

**Lưu ý**: "Hiện đại" đội bóng trang web là không thể nhìn thấy trong các quản trị viên SharePoint **Classic** UI. Bạn có thể truy cập danh sách các trang web "hiện đại" từ [Trung tâm quản trị Sharepoint mới](https://docs.microsoft.com/sharepoint/get-started-new-admin-center).
