# <a name="activating-windows-10-subscriptions"></a>Kích hoạt đăng ký Windows 10

Microsoft 365 Business Premium, Microsoft 365 Education, Microsoft 365 và Enterprise cung cấp giấy phép kỹ thuật số cho Windows 10 như một phần của đăng ký. Để kích hoạt đăng ký, thiết bị của bạn phải đã kích hoạt Windows 10 Pro (1703 hoặc mới hơn). Để xem danh sách đầy đủ các yêu cầu để kích hoạt đăng ký Windows 10, xem [yêu cầu kích hoạt đăng ký Windows 10](https://docs.microsoft.com/windows/deployment/windows-10-subscription-activation#requirements).
