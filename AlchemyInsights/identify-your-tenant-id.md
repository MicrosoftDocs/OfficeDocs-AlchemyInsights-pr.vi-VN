# <a name="identify-your-tenant-id"></a>Xác định ID đối tượng thuê của bạn

Tìm ID tổ chức Microsoft 365 của bạn (số người thuê nhà) trong Trung tâm quản trị Azure.

1. Từ [trang thuộc tính](https://aka.ms/AzurePropertiesPage), hãy đi tới hộp **ID thư mục** .

2. Sao chép ID nếu cần.

Để biết thêm thông tin, hãy xem [Tìm ID tổ chức Microsoft 365 của bạn](https://docs.microsoft.com/onedrive/find-your-office-365-tenant-id).
