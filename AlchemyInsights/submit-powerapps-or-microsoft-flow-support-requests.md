# <a name="submit-powerapps-or-microsoft-flow-support-requests"></a>Gửi yêu cầu hỗ trợ PowerApps hoặc Microsoft Flow

Nếu bạn là khách hàng **PowerApps** hoặc **Microsoft Flow** , vui lòng gửi vé của bạn thông qua [Trung tâm quản trị nền tảng điện](https://admin.powerplatform.microsoft.com/support?newTicket&product=15819)mới. Cổng mới cung cấp đường dẫn trực tiếp hơn để hỗ trợ giúp được tối ưu hóa cho PowerApps và các sản phẩm nền tảng điện khác.

[Gửi yêu cầu hỗ trợ Microsoft Flow](https://admin.powerplatform.microsoft.com/support?newTicket&product=Flow)
