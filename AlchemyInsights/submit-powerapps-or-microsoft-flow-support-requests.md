# <a name="submit-powerapps-or-microsoft-flow-support-requests"></a>Gửi yêu cầu hỗ trợ PowerApps hoặc Microsoft Flow

Nếu bạn là một **PowerApps** hoặc **Microsoft lưu lượng** khách hàng xin vui lòng gửi vé của bạn thông qua mới [nền tảng quyền lực trung tâm quản trị](https://admin.powerplatform.microsoft.com/support?newTicket&product=15819). Cổng mới cung cấp một đường dẫn trực tiếp hơn để hỗ trợ với sự hỗ trợ là tối ưu cho PowerApps và các sản phẩm nền tảng quyền lực khác.

[Gửi yêu cầu hỗ trợ Microsoft Flow](https://admin.powerplatform.microsoft.com/support?newTicket&product=Flow)
