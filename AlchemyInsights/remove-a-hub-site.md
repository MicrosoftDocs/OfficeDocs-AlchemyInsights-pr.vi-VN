# <a name="remove-a-hub-site"></a>Xóa trang web trung tâm

Để huỷ đăng ký một trang web trung tâm quản trị SharePoint mới:
  
1. Trong Trung tâm quản trị SharePoint mới, trong **trang web**, bấm vào **các trang web đang hoạt động**.

2. Chọn trang web, bấm vào **trang web trung tâm**, và sau đó nhấp vào **Unregister như Trung tâm trang web**.

3. Bấm **OK**.

[Tìm hiểu thêm về các trang web trung tâm](https://support.office.com/article/what-is-a-sharepoint-hub-site-fe26ae84-14b7-45b6-a6d1-948b3966427f)
