# <a name="configuring-migration-manager"></a>Cấu hình trình quản lý di chuyển

Trình quản lý di chuyển tập trung quản lý lớn chia sẻ tệp di chuyển bằng cấu hình một hoặc nhiều máy tính hoặc máy ảo (VM) là di trú "đại lý." Để biết thêm thông tin, xem:

- [Thiết lập trình quản lý di chuyển khách hàng](https://docs.microsoft.com/sharepointmigration/mm-setup-clients)

- [Cài đặt trình quản lý di chuyển](https://docs.microsoft.com/sharepointmigration/mm-settings)
