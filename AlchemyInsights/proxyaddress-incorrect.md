# <a name="proxyaddress-incorrect"></a>ProxyAddress không chính xác

Khi một đối tượng được đồng bộ hoá Azure quảng cáo, các giá trị được chỉ định trong thuộc tính proxyAddresses trong Active Directory được so sánh với quy tắc Azure quảng cáo, và sau đó các thuộc tính proxyAddresses là dân cư Azure quảng cáo. Vì vậy, các giá trị của thuộc tính proxyAddresses cho đối tượng Active Directory không thể giống như các giá trị của thuộc tính proxyAddresses ở Azure quảng cáo.
  
Để tìm hiểu thêm về cách proxyaddress là dân cư, hãy xem [làm thế nào các thuộc tính proxyAddress là dân cư Azure quảng cáo](https://support.microsoft.com/help/3190357/how-the-proxyaddresses-attribute-is-populated-in-azure-ad).
