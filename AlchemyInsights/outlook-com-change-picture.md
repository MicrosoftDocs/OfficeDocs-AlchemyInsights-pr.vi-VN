# <a name="change-your-microsoft-account-picture"></a>Thay đổi ảnh tài khoản Microsoft của bạn

1. Đi đến [thông tin tài khoản Microsoft của bạn](https://go.microsoft.com/fwlink/p/?linkid=860841).
2. Chọn **thay đổi ảnh**.
3. Chọn **ảnh mới**.
4. Chọn ảnh bạn muốn sử dụng, rồi chọn **mở**.
5. Kéo hoặc đổi kích cỡ hộp hình ảnh, rồi chọn **lưu**.

Vui lòng cho phép tối đa 24 giờ để thay đổi ảnh hồ sơ của bạn để có hiệu lực.
