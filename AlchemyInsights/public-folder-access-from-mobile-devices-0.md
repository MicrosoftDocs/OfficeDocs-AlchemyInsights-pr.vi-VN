# <a name="public-folder-access-from-mobile-devices"></a>Truy cập thư mục công cộng từ thiết bị di động

Thư mục công cộng không thể truy cập từ thiết bị di động.
  
Nếu bạn muốn có chức năng tương tự như thư mục công cộng và truy cập nó từ thiết bị di động, [hãy xem Tìm hiểu về Microsoft 365 nhóm để](https://support.office.com/article/learn-about-office-365-groups-b565caa1-5c40-40ef-9915-60fdb2d97fa2) có một lựa chọn tốt hơn.
