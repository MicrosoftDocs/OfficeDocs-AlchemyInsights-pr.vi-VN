# <a name="public-folder-access-from-mobile-devices"></a>Truy nhập thư mục công cộng từ thiết bị di động

Các thư mục công cộng không thể truy nhập được từ thiết bị di động.
  
Nếu bạn muốn có các chức năng tương tự như các thư mục công cộng và truy nhập nó từ thiết bị di động, hãy xem [Tìm hiểu về các nhóm Microsoft 365](https://support.office.com/article/learn-about-office-365-groups-b565caa1-5c40-40ef-9915-60fdb2d97fa2) để có thêm lựa chọn tốt hơn.
