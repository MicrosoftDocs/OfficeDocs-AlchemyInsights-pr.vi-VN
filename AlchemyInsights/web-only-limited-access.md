# <a name="web-only-limited-access"></a>Web-Only hạn chế truy nhập

Web-Only hạn chế là một biến thể của mức cấp độ quyền truy nhập ' hạn chế ' cho phép người dùng truy nhập vào đối tượng web.

Để biết thêm thông tin, hãy xem [mức cấp phép mặc định](https://docs.microsoft.com/sharepoint/understanding-permission-levels#default-permission-levels).
