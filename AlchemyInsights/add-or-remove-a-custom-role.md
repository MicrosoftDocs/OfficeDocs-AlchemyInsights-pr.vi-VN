Để quản lý tài nguyên Azure, hãy xem [Thêm hoặc loại bỏ các gán vai trò Azure bằng cổng thông tin Azure](https://docs.microsoft.com/azure/role-based-access-control/role-assignments-portal).

Để quản lý các vai trò tùy chỉnh cho Azure Active Directory (Azure AD), hãy xem [vai trò người quản trị tùy chỉnh trong Azure Active Directory (bản xem trước)](https://docs.microsoft.com/azure/active-directory/users-groups-roles/roles-custom-overview).
