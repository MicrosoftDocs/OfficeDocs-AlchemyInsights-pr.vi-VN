# <a name="use-microsoft-edges-filter-format-for-url-list-based-policies"></a>Sử dụng định dạng bộ lọc của Microsoft Edge cho các chính sách dựa trên danh sách URL

Trong Microsoft Edge versions 77 và phiên bản mới hơn, định dạng bộ lọc có thể được sử dụng cho [danh sách URL – chính sách dựa trên](https://go.microsoft.com/fwlink/?linkid=2135179).
